# VEFÞ1VGA05 

### Áfangageymsla - _Classroom repository_

* Þú getur búið til, breytt og eytt gögnum í áfangageymslunni en geymslan sjálf er eign Tækniskólans.
* Þú getur hlaðið upp verkefnum (_upload files_) beint í áfangageymsluna en það er skynsamlegt að nota þær lausnir sem Github býður upp á. 
* Til að vinna skipulega í verkefnum þarf að vinna staðbundið í áfangageymslunni
* Náðu í áfangageymsluna með **CLONE** aðferðinni af Github. 
* Aðeins er hægt að skoða vefsíður í vafra (_browser_) staðbundið (_local_) 
* [GIT insetnig (install)](https://vefgrunnur.github.io/verkefnaskil/git_innsetning.html) 
* [Allt um GIT má lesa hér](https://vefhonnun.github.io/verkstjorn/index.html)
